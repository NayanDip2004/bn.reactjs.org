আপনার প্রয়োজন অনুসারে যেটুকু দরকার ঠিক সেটুকু React ব্যবহার করতে পারবেন।

React এমন ভাবে নকশা করা হয়েছে যাতে আপনি ধীরে ধীরে এটাকে আয়ত্ত করতে পারেন এবং **আপনার প্রয়োজন অনুসারে যেটুকু দরকার ঠিক সেটুকু React ব্যবহার করতে পারেন।** আপনি যদি কেবল আপনার বর্তমান কোন ওয়েবপাতায় সামান্য কিছু নজরকাঁড়া ইন্টারেক্টিভিটি যোগ করতে চান তাহলে React কম্পোনেন্ট আপনার জন্য অন্যতম সমাধান।

অধিকাংশ ওয়েবসাইটের ক্ষেত্রেই সেগুলো এক পাতার অ্যাপ্লিকেশান হওয়া প্রয়োজনীয় নয়। শুধুমাত্র **কয়েক লাইন কোড লিখে এবং বিশাল সরঞ্জামের চেইন ছাড়াই** আপনার ওয়েবসাইটের ছোট কোন অংশে React ব্যবহার করতে পারেন। অতঃপর আপনার প্রয়োজন মত ধীরে ধীরে এর ব্যবহার বৃদ্ধি করতে পারেন অথবা অল্প কিছু উইজেট হিসেবেই এর ব্যবহার সীমাবদ্ধ রাখতে পারেন।

## এক মিনিটে React যুক্ত করুন {#add-react-in-one-minute}

এই অনুচ্ছেদে আমরা আপনাকে দেখাবো কিভাবে একটি HTML পেজে React কম্পোনেন্ট সংযুক্ত করা যায়। আপনি চাইলে আপনার বর্তমান ওয়েবসাইটটিতেই আমাদেরকে অনুসরণ করে প্রশিক্ষণ নিতে পারেন অথবা একটি নতুন HTML পাতা বানিয়ে প্রশিক্ষণ নিতে পারেন।

এক্ষেত্রে কোন জটিল সরঞ্জাম ইন্সটল করার ঝামেলা নেই -- **এই অনুচ্ছেদটি সম্পূর্ণ করতে প্রয়োজন শুধুমাত্র একটি ইন্টারনেট সংযোগ এবং আপনার মূল্যবান এক মিনিট সময়।**

ঐচ্ছিকঃ [পুরো উদাহরণটি ডাউনলোড করুন (২ কিলোবাইট জিপ ফাইল)](https://gist.github.com/gaearon/6668a1f6986742109c00a581ce704605/archive/87f0b6f34238595b44308acfb86df6ea43669c08.zip)

### ধাপ-১: HTML এ একটি DOM কন্টেইনার যুক্ত করুন {#step-1-add-a-dom-container-to-the-html}

প্রথমত, যেই HTML পাতাটি পরিবর্তন করতে চান সেটি খুলুন। একটা খালি `<div>` ট্যাগ যুক্ত করুন, যেখানে আপনার React এর কার্যকারিতা প্রদর্শন করতে চান। উদাহরণস্বরূপঃ -

```html{3}
<!-- ... existing HTML ... -->

<div id="like_button_container"></div>

<!-- ... existing HTML ... -->
```

আমরা এখানে `<div>` টাকে একটি ইউনিক HTML `id` অ্যাট্রিবিউট দিয়েছি। এই আইডিটি পরবর্তীতে জাভাস্ক্রিপ্ট কোডের ভিতর থেকেও `<div>` টাকে খুঁজে পেতে সাহায্য করবে এবং React কম্পোনেন্ট প্রদর্শন করার কাজে ব্যবহার করা যাবে।

>পরামর্শ
>
>আপনি `<body>` ট্যাগের ভিতর **যেকোনো জায়গায়** একটি "কন্টেইনার" `<div>` ব্যবহার করতে পারেন। আপনার একটা পাতার মধ্যে যত স্বাধীন DOM কন্টেইনার প্রয়োজন ততই ব্যবহার করতে পারেন। এইগুলো প্রাথমিক ভাবে খালি থাকে, পরবর্তীতে React এই DOM কন্টেইনার গুলোর কন্টেন্ট পরিবর্তন করে থাকে।

### ধাপ-২ঃ স্ক্রিপ্ট ট্যাগগুলো সংযুক্ত করুন {#step-2-add-the-script-tags}

এখন আপনার HTML পাতার মধ্যে তিনটি `<script>` ট্যাগ যুক্ত করুন `</body>` ট্যাগ শেষ করার পূর্বেঃ 

```html{5,6,9}
  <!-- ... other HTML ... -->

  <!-- Load React. -->
  <!-- Note: when deploying, replace "development.js" with "production.min.js". -->
  <script src="https://unpkg.com/react@18/umd/react.development.js" crossorigin></script>
  <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js" crossorigin></script>

  <!-- Load our React component. -->
  <script src="like_button.js"></script>

</body>
```

এখানে প্রথম দুটো ট্যাগ React লোড করছে।। তৃতীয়টা আপনার তৈরি করা React কম্পোনেন্ট লোড করছে।

### ধাপ-৩ঃ একটি React কম্পোনেন্ট তৈরি করুন {#step-3-create-a-react-component}

আপনার HTML পেজের ওই একই ডিরেক্টরিতে `like_button.js` নামে একটি ফাইল তৈরি করুন।

এখন **[এই স্টার্টার কোডটি খুলুন](https://cdn.rawgit.com/gaearon/0b180827c190fe4fd98b4c7f570ea4a8/raw/b9157ce933c79a4559d2aa9ff3372668cce48de7/LikeButton.js)** এবং কপি করে আপনার সদ্য তৈরি করা `like_button.js` ফাইলে পেস্ট করুন।

>পরামর্শ
>
>এই কোডটিতে একটি `LikeButton` তৈরি করা আছে। আপনি যদি এই মুহূর্তে কোডটা বুঝতে না পারেন, তবে ভয় পাওয়ার কোন কারণ নেই। আমরা React এর কম্পোনেন্ট কিভাবে তৈরি করা হয় তা [হাতে কলমে টিউটোরিয়াল সিরিজে](/tutorial/tutorial.html) এবং [মেইন কনসেপ্ট সহায়িকায়](/docs/hello-world.html) শিখাবো। এই মুহূর্তে চলুন দেখে আসি স্ক্রিনে কেমন দেখায়।

এখন **[এই শুরু করার কোড](https://gist.github.com/gaearon/0b180827c190fe4fd98b4c7f570ea4a8/raw/b9157ce933c79a4559d2aa9ff3372668cce48de7/LikeButton.js)** এর সাথে `like_button.js` ফাইলের শেষে নিচের তিন লাইন কোড সংযুক্ত করুনঃ

```js{3,4,5}
// ... the starter code you pasted ...

const domContainer = document.querySelector('#like_button_container');
const root = ReactDOM.createRoot(domContainer);
root.render(e(LikeButton));
```

এই তিন লাইন কোড HTML পেজের কাঙ্ক্ষিত `<div>` ট্যাগটি খুঁজে বের করবে, একটি React অ্যাপ তৈরি করবে এবং সেখানে আমাদের "Like" বাটনের জন্যে তৈরি React কম্পোনেন্টটি দেখাবে।

### পরামর্শঃ প্রোডাকশনের জন্য জাভাস্ক্রিপ্টকে ছোটননিত  করুন {#tip-minify-javascript-for-production}

আপনার ওয়েবসাইটের উৎপাদন ও ছুড়ে দেওয়ার আগে অবশ্যই আপনার জাভাস্ক্রিপ্ট কোডসমূহ ছোটননিত করে ফেলুন। কারণ অছোটননিত জাভাস্ক্রিপ্ট আপনার পাতাকে ব্যবহারকারীর নিকট ধীর গতির সেবা দিতে পারে।

আপনি যদি ইতিমধ্যে অ্যাপ্লিকেশন স্ক্রিপ্টসমূহ ছোটননিত করে থাকেন **তাহলে আপনার সাইট এখন উৎপাদনের জন্য প্রস্তুত**। আপনি যদি নির্দিষ্ট করতে চান যে আপনার প্রোডাকশন HTML পেজ React লোড করবে `production.min.js` এই স্ক্রিপ্ট, তাহলে নিচের মত স্ক্রিপ্ট যোগ করুন।

```js
<script src="https://unpkg.com/react@18/umd/react.production.min.js" crossorigin></script>
<script src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js" crossorigin></script>
```

আপনার যদি স্ক্রিপ্টগুলোকে ছোটননিত করার বাবস্থা না থাকে তবে [এই পথ অনুসরণ করতে পারেন](https://gist.github.com/gaearon/42a2ffa41b8319948f9be4076286e1f3)।

## ঐচ্ছিক: React এ JSX ব্যবহার করে দেখুন {#optional-try-react-with-jsx}

উপরের উদাহরণসমূহে আমরা শুধুমাত্র ব্রাউজারে আগে থেকেই সমর্থন করে এমন কোড নিয়ে কথা বলেছি। এই জন্য আমরা জাভাস্ক্রিপ্ট ফাংশন ব্যবহার করেছি React কে জানানোর জন্যে কি দেখাতে হবেঃ

```js
const e = React.createElement;

// Display a "Like" <button>
return e(
  'button',
  { onClick: () => this.setState({ liked: true }) },
  'Like'
);
```

যাইহোক, আরও একটি উপায়ে React ব্যবহার করা যায়, তা হলো - [JSX](/docs/introducing-jsx.html):

```js
// Display a "Like" <button>
return (
  <button onClick={() => this.setState({ liked: true })}>
    Like
  </button>
);
```

উপরের কোড দুটি একে অপরের আমান। এখানে **JSX একটি [সম্পূর্ণ ঐচ্ছিক বৈশিষ্ট্য](/docs/react-without-jsx.html)**, অনেকেই UI কোড লেখার জন্য JSX কে সহায়ক মনে করেন -- React এবং অন্যান্য লাইব্রেরির ক্ষেত্রেও।

আপনি [এই অনলাইন পরিবর্তকের মাধ্যমে](https://babeljs.io/en/repl#?babili=false&browsers=&build=&builtIns=false&spec=false&loose=false&code_lz=DwIwrgLhD2B2AEcDCAbAlgYwNYF4DeAFAJTw4B88EAFmgM4B0tAphAMoQCGETBe86WJgBMAXJQBOYJvAC-RGWQBQ8FfAAyaQYuAB6cFDhkgA&debug=false&forceAllTransforms=false&shippedProposals=false&circleciRepo=&evaluate=false&fileSize=false&timeTravel=false&sourceType=module&lineWrap=true&presets=es2015%2Creact%2Cstage-2&prettier=false&targets=&version=7.15.7) JSX চেষ্টা করে দেখতে পারেন।

### দ্রুত JSX চেষ্টা করে দেখুন {#quickly-try-jsx}

সবচেয়ে সহজে JSX ব্যবহার করার অন্যতম উপায় হচ্ছে নিচের মত করে আপনার‌ পাতাটিতে একটা `<script>` ট্যাগ যুক্ত করুনঃ 

```html
<script src="https://unpkg.com/babel-standalone@6/babel.min.js"></script>
```

এখন আপনি আপনার যেকোনো `<script>` ট্যাগে `type="text/babel"` অ্যাট্রিবিউটটি যুক্ত করে খুব সহজেই JSX লিখতে পারবেন। [এখানে একটি উদাহরণের কোড দেয়া আছে](https://raw.githubusercontent.com/reactjs/reactjs.org/main/static/html/single-file-example.html)।

এই উপায়টা শেখার জন্যে এবং সহজ ডেমো বানানোর জন্যে উত্তম। তবে, এই পদ্ধতিটা আপনার ওয়েবসাইটকে ধীরগতির করে দেবে **এটি প্রোডাকশনের জন্য কোনভাবেই উপযুক্ত নয়।** আপনি যখন আরও গভীরে যেতে প্রস্তুত তখন শেষের `<script>` ট্যাগটি সরিয়ে ফেলুন যার অ্যাট্রিবিউট হিসেবে `type="text/babel"` ব্যবহার করেছিলেন। নিচের অনুচ্ছেদে আমরা দেখবো কিভাবে JSX অব্যপ্রক্রিয়াজাতকরণকারী ব্যবহার করে সকল `<script>`‌ ট্যাগ সমূহকে একসাথে স্বয়ংক্রিয় ভাবে পরিবর্তন করতে পারি।

### একটি প্রোজেক্টে JSX যুক্ত করুন {#add-jsx-to-a-project}

কোন প্রকল্পে JSX যুক্ত করার জন্য তেমন কোন জটিল বান্ডেলকারী এবং উৎপাদন পরিবেশনকারীর প্রয়োজন নাই। আসলে, JSX সংযুক্ত করা অনেকটা **CSS অব্যপ্রক্রিয়াজাতকারক যুক্ত করার মত**। শুধুমাত্র আপনার কম্পিউটারে [Node.js](https://nodejs.org/) ইন্সটল করা থাকতে হবে।

টার্মিনাল থেকে আপনার প্রকল্প ফোল্ডারের অবস্থানে যান এবং নিচের কমান্ড দুটি চালু করুনঃ

1. **ধাপ-১:** Run `npm init -y` (যদি এটা ফেল করে, [তাহলে এখানে একটু দেখুন](https://gist.github.com/gaearon/246f6380610e262f8a648e3e51cad40d))
2. **ধাপ-২:** Run `npm install babel-cli@6 babel-preset-react-app@3`

>খতিয়
>
>আমরা এখানে **npm ব্যবহার করেছি শুধু JSX অব্যপ্রক্রিয়াজাতকারক ইন্সটল করার জন্যে** এছাড়া আর কিছু আপনার দরকার নেই। React ও অ্যাপলিকেশনের কোড আগের মতই `<script>` ট্যাগসহ আছে, কোন পরিবর্তন ছাড়াই।

অভিনন্দন! আপনি এই মাত্র **উৎপাদন উপযোগী JSX সেটআপ** আপনার প্রকল্পে যুক্ত করেছেন।


### JSX অব্যপ্রক্রিয়াজাতকারক চালু করুন {#run-jsx-preprocessor}

একটি ফোল্ডার বানান `src` নামে এবং নিচের কমান্ড গুলো টার্মিনালে চালু করুনঃ

```console
npx babel --watch src --out-dir . --presets react-app/prod
```

> পরামর্শ
>
> `npx` কোন ভুল কমান্ড নয় -- এটি একটি [প্যাকেজ চালু করার টুল, যা npm এর ৫.২+ সংস্করণগুলোতে আছে](https://medium.com/@maybekatz/introducing-npx-an-npm-package-runner-55f7d4bd282b).
>
> আপনি যদি এরকম একটা ত্রুটির বার্তা পান যে- "You have mistakenly installed the `babel` package", আপনি মনে হয় [আগের পদক্ষেপটি](#add-jsx-to-a-project) ছেড়ে এসেছেন। আগের অংশটি একই ফোল্ডারে চালিয়ে দেখুন, অতঃপর আবার চেষ্টা করুন।

এই কাজ শেষ হবার অপেক্ষায় থাকার প্রয়োজন নেই -- এই কমান্ডটি একটি স্বয়ংক্রিয় দর্শনকারী চালু করে দেবে JSX এর জন্যে।

আপনি যদি এখন `src/like_button.js` নামে একটি ফাইল বানান এই **[JSX শুরু করার কোডটি](https://cdn.rawgit.com/gaearon/c8e112dc74ac44aac4f673f2c39d19d1/raw/09b951c86c1bf1116af741fa4664511f2f179f0a/like_button.js)**, দিয়ে। দর্শনকারীটি একটা অব্যপ্রক্রিয়াজাতকৃত `like_button.js` নামে একটা ফাইল বানাবে যাতে সাধারন জাভাস্ক্রিপ্ট কোড লেখা থাকবে, যেন ব্রাউজাররা বুঝতে পারে। আপনি যদি উৎস ফাইলে কোন কিছু পরিবর্তন করেন, তবে তা স্বয়ংক্রিয়ভাবে নতুন করে চালু হবে।

এক্ষেত্রে বোনাস হচ্ছে, এই পদ্ধতিতে আপনি কোন রকম সমস্যা ছাড়াই যেকোন পুরাতন ব্রাউজারের জন্যও মডার্ন জাভাস্ক্রিপ্ট এর সিনট্যাক্স ও বৈশিষ্ট্যসমূহ ব্যবহার করতে পারবেন। আমরা এখানে যেই সরঞ্জামতি ব্যবহার করছি তাকে `Babel` বলা হয়। আপনি এ বিষয়ে বিস্তারিত জানতে চাইলে [এই নথিটা পড়তে পারেন](https://babeljs.io/docs/en/babel-cli/)।

আপনি যদি মনে করেন, এখন আপনি মোটামুটি নির্মাণের সরঞ্জামগুলোতে স্বাচ্ছন্দ্যবোধ করছেন এবং নিজের প্রকল্পের প্রয়োজনে ব্যবহার করতে পারবেন। [পরবর্তী অনুচ্ছেদে](/docs/create-a-new-react-app.html) বেশ কিছু জনপ্রিয় সরঞ্জামচেইন নিয়ে আলোচনা করা হয়েছে। যদি তা না চান, তবে এই স্ক্রিপ্ট ট্যাগগুলো ভালোভাবে কাজ করবে।
